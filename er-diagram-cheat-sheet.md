# ER Diagram Cheat Sheet

## Entity

Basic entity.

```uml
@startuml
    entity foo {
        foo_id
    }
    entity bar {
        bar_id
    }
@enduml
```

In case of using space in entity name.

```uml
@startuml
    entity "foo foo foo" as foo {
        foo_id
    }
    entity "bar bar bar" as bar {
        bar_id
    }
@enduml
```
